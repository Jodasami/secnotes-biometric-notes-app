# SecNotes – Secure Notes with Biometric Authentication

SecNotes es una aplicación móvil desarrollada en Flutter que permite almacenar notas de forma segura utilizando autenticación biométrica del dispositivo.

La aplicación utiliza las capacidades biométricas del teléfono (huella digital o reconocimiento facial) para proteger el acceso a la información del usuario.

---

## Tecnologías utilizadas

Mobile
- Flutter
- Dart

Seguridad
- Biometric Authentication

Plataformas
- Android
- iOS (compatible)

---

## Características principales

- Autenticación biométrica (huella digital / Face ID)
- Creación y gestión de notas seguras
- Interfaz móvil simple y rápida
- Protección de información sensible
- Almacenamiento local seguro

---

## Funcionalidades

- Crear notas
- Editar notas
- Eliminar notas
- Desbloqueo mediante biometría
- Protección de acceso a la aplicación
